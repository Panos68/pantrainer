# PanTrainer

Personal training management app. Log workouts, export to Claude, import AI-generated weekly plans.

## Setup

1. `npm install`
2. Copy `.env.local.example` to `.env.local` and fill in Notion credentials (optional)
3. `npm run dev` — opens at http://localhost:3000

## Data

All training data is stored locally in `data/`. Export files go to `exports/`.
